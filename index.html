Linguist Assessment and Monitoring Environment

LAME 1.2

Installation


1. Drag above button to bookmarks/favorites bar.


2. Go to a supported page.

(Monitoring, Objections, Flitto API, translators.to)


3. Click on LAME button you just dragged.

Usage for Monitoring


1. Select text in any submission (Target, not source).


2. Grade translations.


3. If graded "bad," hover on highlighted text to see your comment.


4. Click ⚡ Copy Revisions ⚡.


5. Paste on Google Sheets (Reviewer's name might take a while to update).

Usage for Objections


1. Press button again to toggle between CHARACHTER and WODRWORD highlighting.
